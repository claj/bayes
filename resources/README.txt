## Datasets

### Pima Indians Diabetes

source:  https://archive.ics.uci.edu/ml/datasets/Pima+Indians+Diabetes

A numeric dataset of biometric data (e.g. age, weight, blood measurements...)
and whether someone had diabetes or not.

### 20 Newsgroups

source: http://web.ist.utl.pt/acardoso/datasets/

These stemmed 20 newsgroups train and test files are cleaned up versions of the
data in the 20_newsgroups folder.

They have one line per document with:

<class><tab><space-separated-tokens ...>

The claimed results get about 0.8103 accuracy using a Naive Bayes classifier on
this dataset.


### Twitter Sentiment Analysis

source: http://thinknook.com/twitter-sentiment-analysis-training-corpus-dataset-2012-09-22/

The twitter dataset has 1,578,627 classified tweets that have a 1 for positive and 0 for negative
sentiment.  They claim about 75% accuracy with a naive bayes classifier.

### SMS Spam collection

source: https://archive.ics.uci.edu/ml/datasets/SMS+Spam+Collection#

The SMS spam collection is a set of SMS tagged messages. It contains lines of
SMS text that start with the class of either ham or spam.  It has a total of
4,827 ham messages (86.6%) and 747 (13.4%) spam messages.
